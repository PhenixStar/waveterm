Presets let you save a Terminolgy layout — including open widgets, connections, and settings — and restore it later.

## Creating a Preset

1. Set up your workspace as desired
2. Go to **File → Save Preset** or run:

```bash
tsh preset save my-dev-setup
```

## Restoring a Preset

```bash
tsh preset load my-dev-setup
```

## AI Presets

AI presets let you save combinations of AI model, system prompt, and widget context settings for different workflows.
