Terminolgy uses a YAML-based configuration file located at `~/.config/terminolgy/settings.yaml`.

## Opening the Config File

```bash
tsh edit ~/.config/terminolgy/settings.yaml
```

Or use the menu: **Terminolgy → Settings**.

## Key Bindings

Terminolgy supports customizable keyboard shortcuts. Common defaults:

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New tab | `Cmd+T` | `Ctrl+T` |
| New block | `Cmd+N` | `Ctrl+N` |
| Close block | `Cmd+W` | `Ctrl+W` |
| Magnify block | `Cmd+M` | `Ctrl+M` |
| Toggle AI panel | `Cmd+Shift+A` | `Ctrl+Shift+A` |
| Navigate blocks | `Ctrl+Shift+Arrow` | `Ctrl+Shift+Arrow` |
