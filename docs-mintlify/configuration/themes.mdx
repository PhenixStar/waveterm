Terminolgy supports light and dark modes, custom themes, and tab background colors.

## Dark and Light Mode

Terminolgy respects your system's color scheme preference by default. You can override this in Settings.

## Tab Background Colors

Right-click any tab header to set a background color or gradient for that tab. This helps visually distinguish different workspaces.

## Custom CSS

Advanced users can inject custom CSS via the settings file for deeper UI customization.
