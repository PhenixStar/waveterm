## Prerequisites

- Node.js 20+
- Go 1.22+
- npm or yarn

## Clone and Install

```bash
git clone https://github.com/PhenixStar/terminolgy.git
cd terminolgy
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```
