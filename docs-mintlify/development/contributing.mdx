We welcome contributions of all kinds — bug reports, feature requests, documentation improvements, and code.

## Reporting Issues

- **Bug reports**: [GitHub Issues](https://github.com/PhenixStar/terminolgy/issues)
- **Feature requests**: [GitHub Issues → Feature Request](https://github.com/PhenixStar/terminolgy/issues/new/choose)
- **Questions**: [Discord community](https://discord.gg/XfvZ334gwU)

## Contributing Code

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a pull request

## Contributing Docs

The docs are open-source. Click "Edit this page" on any docs page to go directly to the GitHub editor for that document.
