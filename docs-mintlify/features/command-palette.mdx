The command palette gives you instant keyboard access to any Terminolgy feature.

## Opening the Command Palette

Press `Cmd+P` (macOS) or `Ctrl+P` (Windows/Linux).

## What You Can Do

- Open new widgets
- Switch between tabs and workspaces
- Run tsh commands
- Access settings
- Search documentation
