Terminolgy includes a built-in system information widget that displays real-time metrics.

## Opening System Info

```bash
tsh view sysinfo
```

Or click the system info icon in the widget bar.

## Available Metrics

- CPU usage (per core and aggregate)
- Memory usage
- Disk I/O
- Network traffic
- Process list
