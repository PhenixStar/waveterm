Terminolgy's web view widget lets you browse web pages and documentation without leaving your terminal.

## Opening a Web Page

```bash
# Open a URL in a new web block
tsh web open https://github.com

# Search the web
tsh web open "terminolgy documentation"

# Open localhost
tsh web open http://localhost:3000
```

## Features

- Full browser rendering (Chromium-based)
- Navigate with back/forward buttons or keyboard shortcuts
- AI can capture screenshots of web widgets for analysis (with Widget Context enabled)
- Terminolgy AI can navigate web widgets programmatically
