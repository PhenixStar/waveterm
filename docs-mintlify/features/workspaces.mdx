Workspaces let you group related tabs and settings under a single project context.

## Creating a Workspace

Access workspaces from the workspace selector in the top-left of the Terminolgy window.

## Workspace Scoped Variables

Use `tsh setvar` with workspace scope to keep variables isolated to a project:

```bash
tsh setvar -b workspace DEPLOY_ENV=staging
tsh getvar -b workspace DEPLOY_ENV
```
