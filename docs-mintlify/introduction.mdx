# Welcome to Terminolgy

Terminolgy is an [open-source](https://github.com/PhenixStar/terminolgy) terminal that combines traditional terminal features with graphical capabilities like file previews, web browsing, and AI assistance. It runs on macOS, Linux, and Windows.

Modern development involves constantly switching between terminals and browsers — checking documentation, previewing files, monitoring systems, and using AI tools. Terminolgy brings these graphical tools directly into the terminal, letting you control them from the command line.

Check out [Getting Started](/quickstart) for installation instructions.
